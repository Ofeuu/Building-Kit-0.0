This system is only functional in Roblox Studio.

The **Client** you need to put in StarterGui.
The **Server** you need to put in ServerScriptService.
The **Objects(ReplicatedStorage)** and Remotes parent both in Workspace.
The **Gui** need to put in StarterGui.
The **Area** and **Objects(Workspace)** parent both in Workspace.

The scripts are in lua and it can be used as an template if scripted correctly.
